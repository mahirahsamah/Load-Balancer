Detailed Description

The Request class defines a request.

Defines an object Request that contains an incoming IP address (ip_in), an outgoing IP address (ip_out), and a time (time) needed for the Request to be processed.

Constructor & Destructor Documentation

◆ request()

request::request ( )

Request object constructor.

A Request object has an incoming IP address that is generated by generate_IP_in(), an outoing IP address that is generated by generate_IP_out(), and a processing time that is generated by generate_time().
